# 🎯 영어학습 서비스 VOC 데이터 분석

> 2026 링글 공모전 | 10개 서비스 7,604건 VOC 분석

---

## 📌 프로젝트 개요

국내 주요 영어학습 서비스 10개의 사용자 리뷰/후기 데이터를 수집하고 분석하여,
**고객이 겪는 진짜 문제**를 데이터 기반으로 도출한 프로젝트입니다.

### 핵심 질문
- 영어학습 서비스 사용자들은 어떤 문제를 겪고 있는가?
- 링글의 강점과 약점은 무엇인가?
- 경쟁사 대비 링글의 포지션은?
- 개선해야 할 우선순위는?

```bash

# 2. 대시보드 (웹 브라우저)
```

## 📈 분석 방법론

### 1. 데이터 수집
- 앱스토어/플레이스토어 리뷰
- 네이버 블로그 후기
- 커뮤니티 (블라인드 등)
- SNS (유튜브 댓글)

### 2. 분석 기법
- 감성 분석 (Sentiment Analysis)
- 키워드 빈도 분석
- TF-IDF 특징 추출
- 페인포인트 분류
- 고객 여정 매핑

### 3. 검증
- 부정 리뷰 전수 분석
- 실제 원문 텍스트 인용
- 데이터 출처 명시
