## Level 1: 시장 전체 조감

### 목적
> "영어 학습 시장 전체가 무슨 이야기를 하고 있는가?"

### 분석 항목

| 분석 | 방법 | 기대 결과 | 상태 |
|------|------|----------|------|
| 키워드 빈도 분석 | 정규식 → 명사 추출 → 빈도 계산 | Top 50 키워드, 워드클라우드 | ✅ 완료 |
| LDA 토픽 모델링 | 전체 리뷰 → 5~7개 토픽 추출 | 토픽별 대표 키워드 | ⏳ 미완료 |
| 페인포인트 랭킹 | 부정 리뷰 → 카테고리 분류 | 불만 카테고리 Top 10 | ⏳ 미완료 |

### 핵심 질문 & 답변
| 질문 | 답변 |
|------|------|
| 가장 많이 언급하는 키워드는? | 영어(5,855), 토익(1,381), 회화(825) |
| 시장의 공통 주제는? | 시험준비, 회화, 튜터품질, 가격 |
| 가장 큰 불만 요인은? | 앱 오류/속도, 가격, 예약 어려움 |

## Level 2: 회사별 특성 비교

### 목적
> "각 회사는 어떻게 다른가? 어떤 특색이 있는가?"

### 분석 항목

| 분석 | 방법 | 기대 결과 | 상태 |
|------|------|----------|------|
| TF-IDF 고유 키워드 | 회사별 TF-IDF 차이 계산 | 회사별 차별화 키워드 | ✅ 완료 |
| 감정 분포 비교 | 평점+텍스트 감정 분석 | 회사별 건강도 지표 | ✅ 완료 |
| 토픽 분포 비교 | Level 1 토픽 × 회사 | 히트맵 | ⏳ 미완료 |
| 포지셔닝 맵 | 2축 매트릭스 | 경쟁 구도 시각화 | ⏳ 미완료 |

### 핵심 질문 & 답변
| 질문 | 답변 |
|------|------|
| 링글만의 고유 키워드는? | 명문대, 튜터, 화상영어, 커리어, 블라인드 |
| 가장 부정적 반응 많은 회사는? | Pagoda(40.3%), Yanadu(45.7%) |
| 링글 vs 경쟁사 차이는? | 링글=프리미엄 커리어, 스픽=AI 무제한 |

## Level 3: 심층 인사이트

### 목적
> "왜 그런가? 링글에게 어떤 의미인가?"

### 분석 항목

| 분석 | 방법 | 기대 결과 | 상태 |
|------|------|----------|------|
| 동시출현 네트워크 | 키워드 간 co-occurrence | 연관 키워드 그래프 | ✅ 완료 |
| 이탈/전환 패턴 | 경쟁사 언급 + 전환 표현 탐지 | 전환 이유 분석 | ✅ 완료 |
| 숨겨진 니즈 | 니즈 패턴 + 키워드 탐지 | 새로운 기회 영역 | ✅ 완료 |
| 세그먼트 분석 | 사용자 유형별 분류 | 타겟별 페인포인트 | ⏳ 미완료 |

### 핵심 질문 & 답변
| 질문 | 답변 |
|------|------|
| "가격"과 함께 언급되는 키워드는? | 시간(331), 추천(280), 수업(280) |
| 스픽에서 링글로 온 이유는? | 실제 대화 경험, 피드백 품질 |
| 새로운 니즈는? | 발표연습(452), 면접영어(288), 회의영어(236) |

## 분석 도구 및 기술

```python
# 실제 사용 라이브러리
- pandas: 데이터 처리
- re (정규식): 한국어 키워드 추출
- sklearn.TfidfVectorizer: TF-IDF 분석
- collections.Counter: 빈도 계산
- networkx: 네트워크 시각화
- matplotlib/seaborn: 차트 시각화
- wordcloud: 워드클라우드
```
