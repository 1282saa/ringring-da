## 핵심 인사이트

### 1. 시험 준비 시장이 크다
- **토익(1,381건)**이 개별 키워드 중 상위권
- 시험준비 카테고리 전체 2,631건으로 1위
- **의미**: 시험 대비 기능 강화 시 큰 시장 접근 가능

### 2. 회화/스피킹 니즈가 강하다
- 회화(825) + 영어회화(836) + 말하기(527) + 스피킹(511) = **2,699건**
- 학습방식_회화 카테고리 2,366건
- **의미**: 발화량 확대가 핵심 가치 ("더 많이 말할 기회")

### 3. 튜터/원어민 의존도 높다
- 튜터(833) + 원어민(690) + 선생님(274) + 강사(155) = **1,952건**
- **의미**: 인적 자원 의존 → AI 대체 기회 존재
- **리스크**: 튜터 품질이 서비스 만족도에 직결

### 4. 가격/결제 관심 높다
- 무료(575) + 할인(519) + 가격(504) + 결제(252) = **1,850건**
- **의미**: 가격 민감 시장, 무료 체험/할인 마케팅 효과적

### 5. AI 인식은 아직 초기
- 인공지능(269) + 자동(102) + 챗봇(49) = **420건**
- 전체 대비 5.5% 수준
- **의미**: AI 기능 자체가 차별점이 되기 어려움, "AI로 뭘 해주느냐"가 중요

### 6. 비즈니스/실무 니즈 존재
- 비즈니스(426) + 업무(187) + 이메일(98) + 직장(78) = **789건**
- **의미**: 직장인 타겟 확대 가능, 회의/발표 준비 기능 니즈
