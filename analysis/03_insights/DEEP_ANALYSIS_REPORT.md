## Part 4: 통계적 검증

### 4.1 분석 방법론

| 분석 | 방법 | 목적 |
|------|------|------|
| 회사간 평점 비교 | One-way ANOVA | 집단간 평균 차이 검정 |
| AI vs 휴먼 비교 | Independent t-test | 두 집단 평균 비교 |
| 긍정률 차이 | Chi-square test | 비율 차이 검정 |
| 신뢰도 | 95% CI | 추정의 정확도 |

### 4.2 주요 통계 결과

#### 4.2.1 회사간 평점 차이 (ANOVA)
```
F-statistic: 98.06
p-value: < 0.001 ***
결론: 회사간 통계적으로 유의미한 평점 차이 존재
```

#### 4.2.2 AI vs 휴먼 튜터 비교 (t-test)
```
AI 기반 서비스:     M = 3.79, n = 1,385
휴먼 튜터 서비스:   M = 4.19, n = 1,872

t = -7.58, p < 0.001 ***
Cohen's d = 0.267 (Small-Medium Effect)

해석: 휴먼 튜터 서비스가 AI 서비스보다 통계적으로
      유의미하게 높은 평점 (효과크기: 소~중간)
```

#### 4.2.3 긍정률 비율 차이 (Chi-square)
```
χ² = 641.77, df = 9, p < 0.001 ***
결론: 회사간 긍정/부정 비율이 통계적으로 유의미하게 다름
```

### 4.3 회사별 95% 신뢰구간

| 회사 | 평균 | 95% CI | 해석 |
|------|------|--------|------|
| **CARROT** | 4.65 | [4.57, 4.74] | 가장 높고 안정적 |
| UPHONE | 4.35 | [4.27, 4.43] | 높고 안정적 |
| HACKERS | 4.34 | [4.25, 4.44] | 높음 |
| MALHAE | 4.34 | [4.23, 4.45] | 높음 |
| CAKE | 4.17 | [4.04, 4.31] | 양호 |
| MAXAI | 4.07 | [3.91, 4.24] | 넓은 CI (불안정) |
| SANTA | 3.37 | [3.25, 3.50] | 보통 이하 |
| **RINGLE** | 3.19 | [2.99, 3.39] | 개선 필요 |
| YANADU | 2.87 | [2.71, 3.02] | 낮음 |
| PAGODA | 1.96 | [1.54, 2.38] | 심각 |

### 4.4 통계적 시사점

1. **휴먼 튜터 우위 확인** (p<0.001)
   - AI 대체 불가 영역 존재
   - 링글의 "휴먼 튜터" 자산 가치 높음

2. **링글 평점 개선 시급**
   - 95% CI 상한(3.39)도 경쟁사 하한 미달
   - 튜터 품질 관리가 핵심 과제

3. **효과크기(d=0.267) 해석**
   - "작은~중간" 수준의 실질적 차이
   - 휴먼 튜터가 무조건 우월하진 않음
   - AI+휴먼 하이브리드 전략 유효

## 전략적 제언

### 링글을 위한 Action Items
