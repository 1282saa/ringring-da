3. 빈도 기준 상위 조합 분석


## 링글 AI를 위한 시사점

### 1. 시간 효율성 = 핵심 가치
- "시간"이 거의 모든 주요 키워드와 연결
- **AI 장점**: 24시간 가능, 예약 불필요, 즉시 시작
- **메시지**: "바쁜 직장인의 시간을 아껴주는 AI 튜터"

### 2. 가격 + 시간 + 효과 삼각 구조
```
     가격
    /    \
   /      \
시간 ────── 효과
```
- 사용자는 "이 가격에 이 시간 투자해서 이만큼 효과 있나?" 계산
- **AI 포지셔닝**: "합리적 가격 × 적은 시간 × 높은 효과"

### 3. 불편 → 앱 + 시간
- 불편함이 앱, 시간과 동시 언급
- **우선 해결**: 앱 안정성, 빠른 로딩, 시간 낭비 최소화

### 4. 토익 시장 진입 기회
- 토익-시험-점수 강한 클러스터
- **AI 기능**: AI 모의고사, 파트별 취약점 분석, 점수 예측
