# Level 2-1: 회사별 TF-IDF 고유 키워드 분석
> 분석일: 2026-01-18
> 데이터: 10개 서비스, 7,604건 유효 리뷰

---

## 분석 개요

### 목적
각 서비스만의 고유한 특성을 나타내는 키워드 추출
→ "이 회사는 타사와 뭐가 다른가?"

### 방법
1. 회사별 전체 리뷰를 하나의 문서로 결합
2. TF-IDF 벡터 계산 (10개 회사 × 키워드)
3. 각 회사의 TF-IDF 점수 - 타사 평균 = **차별화 점수**
4. 차별화 점수 상위 키워드 = **고유 키워드**

## 회사별 포지셔닝 분석

### 1. Ringle - "프리미엄 커리어 영어"
```
고유 키워드: 명문대, 튜터, 화상영어, 커리어, 블라인드, 연봉
```
**해석:**
- **명문대**: 아이비리그/명문대 출신 튜터 강조
- **커리어, 연봉, 블라인드**: 직장인/커리어 목적 사용자 많음
- **화상영어**: 1:1 화상 수업이 핵심 서비스

**포지션:** 고관여 직장인을 위한 프리미엄 1:1 화상영어

---

### 2. Cake - "콘텐츠 기반 표현 학습"
```
고유 키워드: 영상, 표현, 유튜브, 공감, 자료
```
**해석:**
- **영상, 유튜브**: 영상 콘텐츠 기반 학습
- **표현**: 실전 표현 학습에 초점
- **공감**: MZ세대 감성 소통

**포지션:** 영상으로 배우는 실전 영어 표현

---

### 3. Hackers - "토익 시험 전문"
```
고유 키워드: 토익, 시험, 파트, 강의, 실전
```
**해석:**
- **토익, 파트**: 토익 파트별 학습 특화
- **시험, 실전**: 시험 대비 목적 명확
- **강의**: 인강/학원 강의 중심

**포지션:** 토익 시험 대비 전문 (인강+학원)

---

### 4. Malhae (말해보카) - "어휘/발음 게이미피케이션"
```
고유 키워드: 단어, 어휘, 발음, 문법, 스픽
```
**해석:**
- **단어, 어휘**: 어휘력 향상 특화
- **발음, 문법**: 기초 영어 능력
- **스픽**: 경쟁사 언급 (비교 대상)

**포지션:** 게임처럼 배우는 단어/발음 앱

---

### 5. MaxAI - "AI 대화 영어회화"
```
고유 키워드: 대화, 영어회화, 원어민, 듀오링고, 자연스럽게
```
**해석:**
- **대화, 영어회화**: 회화 중심
- **원어민**: 원어민 수준 추구
- **듀오링고**: 글로벌 경쟁사 언급
- **자연스럽게**: 자연스러운 대화 강조

**포지션:** AI 기반 자연스러운 영어 대화

---

### 6. Pagoda - "전통 어학원 + 가격 할인"
```
고유 키워드: 학원, 할인, 가격, 시간, 블로그
```
**해석:**
- **학원**: 오프라인 학원 연계
- **할인, 가격**: 가격 프로모션 많음
- **블로그**: 블로그 마케팅/후기

**포지션:** 전통 어학원의 온라인 확장

---

### 7. Santa - "토익 AI 점수 예측"
```
고유 키워드: 문제, 점수, 환급, 데이터, 파트
```
**해석:**
- **문제, 파트**: 토익 문제 풀이 중심
- **점수**: 점수 예측/향상 목표
- **환급**: 환급 마케팅
- **데이터**: AI/데이터 기반 강조

**포지션:** 데이터 기반 토익 점수 예측 및 맞춤 학습

---

### 8. Uphone (민병철유폰) - "전통 전화영어"
```
고유 키워드: 전화영어, 화상영어, 수업, 시간, 블로그
```
**해석:**
- **전화영어**: 전화 기반 영어 수업
- **화상영어**: 화상으로 확장
- **시간**: 시간 예약/관리

**포지션:** 전통 전화영어 (B2B 강자)

---

### 9. Yanadu - "초보자 강의 + 환급"
```
고유 키워드: 강의, 시원스쿨, 패키지, 하루, 브랜드
```
**해석:**
- **강의**: 강의 콘텐츠 중심
- **시원스쿨**: 주요 경쟁사 언급
- **패키지, 하루**: 패키지 판매, 하루 10분 학습
- **브랜드**: 브랜드 인지도 마케팅

**포지션:** 왕초보 환급 강의 (동기부여 마케팅)

---

### 10. Carrot - "튜터 중심 피드백"
```
고유 키워드: 선생님, 실력, 시간, 화상, 수업
```
**해석:**
- **선생님**: 튜터/강사 중심
- **실력**: 실력 향상 목표
- **화상, 수업**: 화상 수업 방식

**포지션:** 튜터 품질 중심 화상/전화영어

## 링글 AI에 대한 시사점

### 현재 링글의 포지션
- **강점**: 명문대 튜터, 커리어 영어, 프리미엄
- **고유 영역**: 비즈니스/커리어 목적 고관여 사용자

### 경쟁사 대비 차별화 방향

| vs | 링글의 차별점 | AI로 강화할 점 |
|----|-------------|---------------|
| **Cake** | 튜터 피드백 (콘텐츠만 vs 대화) | AI 콘텐츠 추천 |
| **Hackers/Santa** | 회화 중심 (시험 vs 실전) | AI 회화 연습 |
| **MaxAI** | 프리미엄 품질 (대중 vs 전문) | AI 품질 차별화 |
| **Uphone/Carrot** | 명문대 튜터 (일반 vs 엘리트) | AI + 튜터 하이브리드 |
| **Yanadu** | 고급 학습자 (초보 vs 중급+) | AI 레벨 맞춤 |

### 20% 점유율을 위한 확장 방향

1. **시험 시장 진입**: Santa/Hackers 영역 (토익/오픽 AI 모의고사)
2. **초보 시장 진입**: Yanadu 영역 (AI 왕초보 코스)
3. **가격 라인업**: Cake/Malhae 영역 (AI 단독 저가 상품)
