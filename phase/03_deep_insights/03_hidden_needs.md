---

## 분석 개요

### 목적
명시적으로 드러나지 않는 사용자의 잠재 니즈 발굴
→ "사용자가 직접 말하진 않지만 원하는 것은?"

### 방법
1. 니즈 패턴 탐지: "있으면 좋겠다", "왜 없나", "아쉽다" 등
2. 특정 키워드 기반 니즈 분류: 비즈니스 시나리오, 기능 요청 등
3. 불만 표현에서 역으로 니즈 추출

### 1. 비즈니스 영어 니즈 (높음)

#### 발표 연습 (245건)
```
"프리젠테이션 준비에서 큰 도움받았습니다"
"발표 자료 영어로 만들 때 도움"
```
**AI 기회**: AI 발표 연습 모드, 실시간 피드백

#### 면접 영어 (203건)

```
"영어 면접 준비 중인데..."
```
**AI 기회**: AI 면접관 모드, 예상 질문 연습

## 핵심 인사이트

### 1. 비즈니스 영어 = 링글의 핵심 기회
- 발표, 회의, 면접, 이메일 → 총 712건 (링글만)
- **기존 강점 (명문대 튜터) + AI = 비즈니스 영어 최강자**

### 2. 맞춤화 니즈 강함
- 목표 설정(893건), 업종별(804건)
- **"나에게 맞춤화된 학습"이 핵심 가치**

### 3. 콘텐츠 다양화 요구
- 드라마, 뉴스, 취미 영어 니즈 높음
- **AI로 무한 콘텐츠 생성 가능**

## 사용자 세그먼트별 숨겨진 니즈

### 직장인 세그먼트
```
핵심 니즈: 발표 > 회의 > 이메일 > 협상
숨겨진 니즈: "바로 내일 쓸 수 있는 영어"
AI 솔루션: 상황별 즉시 연습 모드
```

### 취준생/이직자 세그먼트
```
핵심 니즈: 면접 > 자기소개 > 경력 설명
숨겨진 니즈: "합격하는 영어 면접"
AI 솔루션: 산업별 면접 시뮬레이션
```

### 초중급 학습자 세그먼트
```
핵심 니즈: 목표 설정 > 체계적 커리큘럼
숨겨진 니즈: "혼자서도 꾸준히 할 수 있게"
AI 솔루션: 일일 미션 + 리마인더
```
